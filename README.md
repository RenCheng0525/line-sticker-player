# LINE Sticker Player

一支**純前端、單一 HTML 檔**的小工具,可以**瀏覽** LINE 商店的貼圖組、**播放**動畫貼圖(APNG)、並整組**下載**成 ZIP 或產生**可離線使用**的版本。沒有後端、沒有建置步驟,打開即用。

> ℹ️ **本 repo 不含可執行檔。** 這裡只放建置規格與說明;實際的 `line-sticker-player.html` 請依 [`CLAUDE.md`](./CLAUDE.md) 自行產生(見下方)。本 repo 也**不含任何貼圖素材**。

## 功能

- **瀏覽**:選一個分類,列出該頁的貼圖組(動畫縮圖、名稱、ID、類型),可翻頁。
- **播放**:點任一組或輸入貼圖組 ID,即時播放整組 APNG 動畫;支援單張 / 全部的暫停、重播,以及大小調整。
- **保存**:整組打包下載 ZIP,或產生一份把動畫全部內嵌的**離線 HTML**——之後打開零連線即可播放/暫停。

## 取得方式:自行產生

本工具刻意**不**以可執行檔散布;請用提示詞自行產生:

1. 把 [`CLAUDE.md`](./CLAUDE.md) 放進一個資料夾。
2. 在該資料夾開 [Claude Code](https://www.anthropic.com/claude-code)(或把內容貼給 Claude),說「**依照 CLAUDE.md 產生這個 app**」。
3. 會得到一支 `line-sticker-player.html`。

## 執行

產生出 `line-sticker-player.html` 後,用瀏覽器打開即可;若遇到字型或抓取問題,改用本機伺服器開啟更穩:

```bash
# 在檔案所在資料夾執行其中一種
python3 -m http.server 8080
# 或
npx serve
```

然後瀏覽 `http://localhost:8080/line-sticker-player.html`。在「瀏覽」分頁挑分類按「載入清單」,點某組「開啟」就會跳到「播放器」播放;或在「播放器」直接輸入 ID 載入。

## 關於跨來源(CORS)

「顯示動畫」用 `<img>`,**不受 CORS 限制**,只要拿得到貼圖 ID 一定播得出來。但「瀏覽清單」「打包下載」需要跨網站讀取內容,會被瀏覽器擋,因此工具會依序嘗試多個公開 CORS 代理。若代理全部失效,可改用內建的**手動貼上**(貼分類頁內容、貼 ID、或貼 `productInfo.meta`),完全不連外也能運作。

公開代理本來就時靈時不靈;若要穩定自動抓取,建議自行架一個簡單的本機反向代理補上 CORS 標頭。

## 原理

想了解運作原理,可參考 [`line-sticker-player-原理.md`](./line-sticker-player-原理.md):涵蓋 CDN 端點、APNG 原生播放、`<img>` 顯示 vs `fetch` 讀取的 CORS 差異、多代理輪詢、播放/暫停與離線版的做法。

## 支援範圍

| 貼圖類型 | 狀態 |
| --- | --- |
| 動態 / 有聲動態 | 動畫可播(聲音目前未播放) |
| 靜態 / 大貼圖 | 退回顯示靜態圖 |
| 全螢幕 / 特效 | 用 `popup` 類資源,目前會退回靜態(可擴充) |
| 訊息 / 隨你填 | 重點為動態合成的使用者文字,抓圖無法重現 |
